# BTech Python

## Prerequisites

- [Cloudflare 1.1.1.1](https://one.one.one.one/)

- Python >= 3.7

## Installation

```bash
python -m pip install -r requirements.txt
```

## Usage

To scrape data, please make sure [1.1.1.1](https://one.one.one.one/) is on, then run

```bash
python 00_scrape.py
```
